# ofxHIDAPI

## notice

* currently supporting macOS only
* maybe win & linux users can use this with install `hidapi` and update `addon_config.mk`

## dependencies

* [libusb/hidapi](https://github.com/libusb/hidapi)

## setup

```
brew install hidapi
```
